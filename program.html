Описание курса

В первой части курса вы познакомитесь со стандартами Web 2.0, научитесь работать с макетами и форматировать кодбосвоите адаптивную версткуб а по итогу-сможете создавать быстрые и удобные сайты, которыеточно понравятся польтзователям.

Вы поймете название основных тегов HTML, сможете без усилий стилизировать и размещать єлементі с помощью CSS и сверстаете свой первый макет.

Вторая часть курса сделает из вас настоящего масмтера. Вы изучите сборщики проектов, научитесь верстать email-рассылки, разберетесь с анимацией сайта.

Помимо этого, вы сверстаете фрагмент из интернет-магазина - главная страницу,страницу каталога и страницу товара!
